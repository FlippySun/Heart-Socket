# 更新日志

本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [0.1.0] - 2026-02-08

### 🎉 首次发布

#### 核心功能
- **实时心率显示** — 状态栏显示当前心率 BPM + 心跳动画
- **9 级心率区间** — 编程场景优化的细粒度心率区间，每个区间对应不同颜色
- **多种健康数据** — 心率、卡路里、步数、血氧、距离、速度、体重、BMI
- **智能告警** — 高心率 / 低心率弹窗提醒，可配置阈值和冷却时间

#### 数据源支持
- **HDS 本地直连** ⭐ — Apple Watch → WiFi → VS Code，零配置零中间件
- **Pulsoid** — 免费云端方案，支持 Apple Watch
- **HypeRate** — 商业 API 方案
- **自定义 WebSocket** — 连接任意心率数据源，支持多字段 JSON Path 提取

#### Motion 传感器分析 (v3 VMUA)
- **⌨️ 敲代码强度** — 基于手腕加速度实时分析打字强度（💤⌨️⚡🔥🚀 五级）
- **🖐️ 5 种姿态识别** — 😴 静息 / ⌨️ 打字 / 🖱️ 鼠标 / 💪 活动 / 🚶 走动
- **🪑 久坐提醒** — 步数 + 加速度综合判断，超阈值弹窗提醒
- **🎯 心流检测** — 稳定打字 + 稳定心率超过 15 分钟自动识别
- **🐟 摸鱼指数** — 综合姿态、打字强度、久坐时长的评分（0-100）
- **🔋 精力水平** — 基于心率趋势、活动量、时段的精力评估
- **编辑器活动回退** — Pulsoid / HypeRate / 自定义数据源无 Motion 时自动回退

#### 统计与历史
- **心率统计面板** — 实时仪表盘：趋势图、区间分布饼图、健康数据总览
- **📅 历史日历** — 日历月视图查看心率日报，持久化存储 90 天

#### 网络与连接
- **自动重连** — 网络断开后指数退避自动重连
- **网络变化检测** — WiFi / IP 变化时自动弹窗提醒更新 Watch 地址
- **端口冲突处理** — 端口被占用时自动提示并引导配置
